Telegram Bot Configuration

How to get your Bot Token and Chat ID:

  1. Bot Token: Message @BotFather on Telegram, create a new bot with /newbot, and copy the token
  2. Chat ID: Message your bot, then visit https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates and find your chat ID in the response

Trade / detector

Midgard polling & opportunity filters:

Minimum size and duration gate valid opportunities. Monitored assets are passed to Midgard asset (OR: swaps involving at least one listed asset). With several assets, each is polled in sequence; add a few hundred ms between calls to limit rate (429) errors. One asset per line or comma-separated.

Test with a historical transaction:

Enter a THORChain transaction hash to fetch it from Midgard and trigger the detection pipeline. This will emit an action.detected event, which will be processed by the DetectorService.
